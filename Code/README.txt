Extras:

- To plot all electric field vector intensities for all identified modes, call
                    plot_intensities_of_vector_fields()

- To plot the directional vector field lines for a certain mode, e.g. 'HE_22', call:
                    choose_E_vector_fields('HE_22')

- To plot the 1-dimensional cut of the electric field, e.g. the 'E_phi' for 'HE_22':
                    plot_1D_E_field('HE_22', 'E_phi')

- To plot all the total intensities for all identified modes, call:
                    plot_intensities_all(plot = True)
